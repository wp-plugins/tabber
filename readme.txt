这是一款有TAB切换效果的侧边栏小工具插件

== Description ==

有TAB切换效果的侧边栏小工具，显示最新文章（10篇），显示最新留言（5条），显示TAG（18个热门TAG）

== Tips ==

请注意，此插件没有后台的CSS管理，需要您手动修改CSS里面的样式。具体请看我的博客的介绍。

== Installation ==

 (1) unzip the tabber.zip file in /wp-content/plugins 解压tabber.zip文件至/wp-content/plugins目录

 (2) Active the plugin 激活插件
